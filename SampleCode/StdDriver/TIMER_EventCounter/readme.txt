Purpose:
---------------------------------------------------------------------------------------------------------
    Demonstrate how to implement use timer controller
    (1) Implement timer1 event counter function to count the external input event.
		
---------------------------------------------------------------------------------------------------------
Operation:
---------------------------------------------------------------------------------------------------------
    (1) Connects to comport to send out demo message (TX=PB8, RX=PB9)
    (2) Connect (PD.4) pin to event counter pin T2(PB.2)
    (3) Compiled to execute.
    (4) Program test procedure -
	 1. pull PD.4 High/Low to generate T2 event input source
         2. The terminal show "Get Timer2 event counter value is 56789 .... PASS"
