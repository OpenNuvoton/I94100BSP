---------------------------------------------------------------------------------------------------------
Purpose:
---------------------------------------------------------------------------------------------------------
    Demonstrate how to use "Clock Fail Detector" to handler clock source failure.
	(1)	Change system clock source from HXT to HIRC, if HXT has failed.
		
---------------------------------------------------------------------------------------------------------
Operation:
---------------------------------------------------------------------------------------------------------
    (1) Connects to com port to send out demo message (TX=PB8, RX=PB9).
    (2) Compiled to execute.
    (3) Program test procedure -
        1. After showing "Stop HXT to test.", remove the HXT oscillator.
		2. When showing "HXT Clock is stopped! HCLK is switched to HIRC.", the demo is finished.
