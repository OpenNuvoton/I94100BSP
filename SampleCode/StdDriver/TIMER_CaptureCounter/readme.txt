---------------------------------------------------------------------------------------------------------

    (2) Connect pin PD.2 <-> PB.2
    (3) Connect pin PC.2 <-> PB.3
    (4) Compiled to execute.
    (5) Program test procedure -
	1. Every 500 event counts will be captured when Time2 capture trigger event occurred.
